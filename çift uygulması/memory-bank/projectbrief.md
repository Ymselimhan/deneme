# Project Brief: Çift Uygulaması (Couples App)

## Overview
A cross-platform mobile application (iOS & Android) built with Flutter, designed for couples to maintain a private shared space. The system will be self-hosted on the user's own server.

## Core Goals
- Provide a private, shared space for couples.
- **Real-time Location Tracking**: Partners can see each other's current location.
- **Instant Messaging**: A private chat feature for the couple.
- Self-hosted backend for maximum privacy and control.

## Scope
- Flutter mobile applications for iOS and Android.
- Custom backend server (self-hosted).
- Secure authentication and partner linking.
- Dashboard with shared widgets.
- Location sharing and history.
- Real-time chat.
